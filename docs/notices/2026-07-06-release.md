# 업데이트 안내 — 버전 관리 강화·신규 화면·에디터 개선 (2026-07-06)

이번 업데이트로 아래 기능이 추가·개선되었습니다. 자세한 사용법은 상단 **사용 매뉴얼**에서 확인하실 수 있습니다.

## 버전 관리가 강화되었습니다

- 게시된 버전에 **버전 번호(v.1, v.2 …)**가 부여됩니다. 새 버전을 게시하면 직전 게시본은 #만료됨 상태로 보존됩니다.
- **점유권(체크아웃)**: 초안은 한 번에 한 사람만 편집합니다. 다른 편집자에게 **점유권 요청**을 보내거나 보유자가 직접 **이전**할 수 있고, 30분 미사용 시 자동 해제됩니다.
- 승인 요청의 **회수**, 게시/만료본에서 새 초안을 만드는 **재게시**를 지원합니다.

## 새 화면 4종이 열렸습니다

| 화면 | 내용 |
| --- | --- |
| **공지사항** | 지금 보고 계신 이 화면 — 중요/일반 필터와 읽음 표시 |
| **알림·승인(Inbox)** | 내가 처리할 승인 요청(버전·점유권·권한)과 알림을 한곳에서 승인/반려 |
| **피드백** | 버그·제안을 관리자에게 바로 전송, 처리 상태와 답변 확인 |
| **사용 매뉴얼** | 한/영 다중 문서 — 제목 드롭다운으로 선택, 목차·본문 검색 지원 |

## 하위프로세스(링크 맵) 사용 방식이 바뀌었습니다

- **지정된 맵만** 라이브러리 피커에 나타납니다. 맵 소유자가 **맵 설정 → 서브프로세스 지정**에서 대표 부서 등 어트리뷰트와 함께 지정하세요.
- 지정 어트리뷰트(부서·담당자·시스템·소요시간)는 그 맵을 링크한 모든 노드에 **라이브로 표시**됩니다.
- 미지정 맵을 가리키는 기존 링크 노드는 경고와 함께 잠기며, 지정하면 다시 풀립니다.

## 에디터가 편해졌습니다

- **자동 정렬 가로/세로**: `Shift+L`(좌→우) / `Shift+K`(상→하) — 시작→대표 끝 주 경로가 한 직선으로 정렬되고 엣지 방향도 함께 맞춰집니다.
- **노드 검색**이 `/` 키 한 번으로 열립니다(초성 검색 지원).
- 단축키 안내는 좌측 사이드바 **단축키 더보기**로 이동했고, 우하단에는 줌 컨트롤이 배치되었습니다.
- **PNG 내보내기**에서 엣지가 누락되던 문제를 수정했습니다 — 모든 연결선이 검은 실선으로, 2배 해상도로 선명하게 출력됩니다.

## 더 빨라졌습니다

- 담당자·협업자 피커와 각종 목록이 **25개씩 끊어 로딩**되어 인원이 많아도 즉시 열립니다.
- 검색 결과 순서가 똑똑해졌습니다 — 단어 시작 일치 우선, 띄어쓰기로 여러 단어 검색(`kim jun`), 부서/그룹 매치는 최상단 고정.
- 맵 상세 정보 로딩 중에는 스피너와 자리표시가 나타나 화면이 덜컹이지 않습니다.

문의나 불편 사항은 상단 **피드백** 버튼으로 보내주세요.
