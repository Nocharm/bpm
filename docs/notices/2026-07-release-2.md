# 업데이트 안내 — 홈 개편·승인 워크플로·서브프로세스 연결 (7월 2차)

이번 업데이트로 아래 기능이 추가·개선되었습니다. 자세한 사용법은 상단 **사용 매뉴얼**(맵 편집 · 사용 안내)에서 확인하실 수 있습니다.

## 홈 화면이 새로워졌습니다

- **좌측 조직도**로 부서별 맵을 탐색합니다 — **나의 부서**는 즐겨찾기로 맨 위에 고정됩니다.
- 홈 우측에 **현황 대시보드**가 생겼습니다 — 맵 상태 도넛 차트, 최근 열어본 맵, 내가 처리할 승인.
- 맵 카드·상세에 **오우닝 부서**와 **SP 배지**(서브프로세스 지정 맵)가 표시됩니다.
- 새 맵에는 **시작·종료 노드가 자동으로** 만들어져 바로 그리기 시작할 수 있습니다.

## 승인 워크플로가 편해졌습니다

- **셀프 게시** — 결재자가 본인 1명이면 승인 요청 시 "바로 게시할까요?" 팝오버로 제출→승인→게시를 **원클릭**으로 끝냅니다.
- **맵 이름 변경 요청** — 편집자가 이름을 바꾸면 오너에게 승인 요청이 가고, 오너는 Inbox에서 승인/반려합니다(오너는 지금처럼 즉시 변경).
- 새 버전을 만들 때 이름을 직접 입력하고, **버전 번호는 게시 시 자동 부여**됩니다.

## 서브프로세스 연결이 유연해졌습니다

- **미등록 맵도 먼저 링크(플레이스홀더)** — 프로세스 라이브러리의 "Show unregistered maps"를 켜면 아직 서브프로세스로 지정되지 않은 맵도 클릭해 링크할 수 있습니다. 링크하면서 대상 맵 오너에게 **등록 요청**을 보내면, 오너가 Inbox에서 지정 폼을 저장하는 것으로 수락됩니다.
- **새 맵을 그 자리에서** — 라이브러리 검색어로 **New map**을 누르면 이름이 채워진 생성 창이 뜨고, 만들자마자 현재 맵에 자동으로 링크됩니다.
- 상단 맵 드롭다운에는 서브프로세스 지정 맵 **SP 배지**와 이미 링크한 맵 **체크 표시**가 붙습니다.
- SP 목록은 우클릭 또는 **`S` 단축키**로 열고(한글 초성 검색), 같은 맵은 두 번 링크되지 않습니다. 새 링크는 기본으로 **최신 게시본을 자동 추종**합니다.
- 지정 폼에 **설명** 항목이 생겼고, 하위프로세스 노드를 선택하면 인스펙터 **Subprocess 탭**에서 지정 정보와 "이 맵을 쓰는 곳" 목록을 볼 수 있습니다.

## 편집이 빨라졌습니다

- **노드 복사/붙여넣기** — `Ctrl+C` / `Ctrl+V`. 다른 탭·다른 맵에도 붙여넣을 수 있습니다. `Ctrl`+드래그로 복제.
- **`Shift`+드래그**로 노드를 한 축(수평/수직)으로만 이동해 줄 맞추기가 쉽습니다.
- **그룹 일괄 편집**에서 회당 파라미터 6개 값도 한 번에 채웁니다.

## 그 밖에

- **Excel 내보내기 형식 2종** — 구조 정리형(분기 주석 포함)과 **WBS형**(하위프로세스를 레벨로 펼침) 중 선택합니다.
- **알림 정리** — 카테고리 필터, 선택 삭제·읽은 알림 삭제·날짜 지정 삭제를 지원하고, 알림은 1인당 최근 100건까지 보관됩니다.
- 사용 매뉴얼에 **At a glance** 드롭다운이 생겨 문서를 빠르게 오갈 수 있습니다.

문의나 불편 사항은 상단 **피드백** 버튼으로 보내주세요.
