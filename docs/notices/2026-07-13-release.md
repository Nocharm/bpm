# 업데이트 안내 — 회당 파라미터·가져오기/내보내기·대시보드 (2026-07-13)

이번 업데이트로 아래 기능이 추가·개선되었습니다. 자세한 사용법은 상단 **사용 매뉴얼**(맵 편집 · 사용 안내)에서 확인하실 수 있습니다.

## 노드에 회당 파라미터가 생겼습니다

각 노드에 프로세스를 **한 번 수행할 때**의 비용·소요를 기록할 수 있습니다.

- **6개 값**: 회당 소요시간(h) · 회당 추가비용(원) · 회당 추가비용($) · 회당 투입인원 · 연간 건수 · FTE. 인스펙터의 파라미터 섹션에서 편집합니다.
- **소요시간은 "시.분"**으로 적습니다 — 소수부는 십진수가 아니라 분입니다(`0.30`=30분, `1.30`=1시간 30분). 화면에는 `1h30m`으로 표시됩니다.
- **비용은 한 통화만** — 원(₩)과 달러($)를 동시에 넣을 수 없습니다. 천단위 콤마는 자동으로 붙습니다.
- **하위프로세스 노드**는 연간 건수·FTE만 직접 입력하고, 소요시간·비용·투입인원은 링크한 맵의 지정값을 읽기전용으로 상속합니다.
- 맵을 서브프로세스로 지정할 때 게시본의 파라미터를 **합산 미리보기**(비용은 합, 투입인원은 평균)로 제안합니다.

## CSV·AI로 맵을 빠르게 채웁니다

- **CSV 가져오기**: 상단 바 **가져오기**에서 CSV를 올려 노드를 채웁니다. **제목으로 병합**하므로 기존 노드의 색·댓글·그룹은 유지되고, 빈 칸은 기존 값을 그대로 둡니다. **템플릿 다운로드**로 빈 양식을 받으세요.
- **CSV로 새 맵 만들기**: 홈의 **New map** 옆 드롭다운에서 **Create from CSV**를 고르면 CSV 파일을 드롭해 새 맵을 바로 만듭니다.
- **다른 AI에게 물어보기**: 가져오기 창에서 프롬프트를 복사해 외부 AI에 문서와 함께 붙여넣고, 받은 CSV를 다시 붙여넣을 수 있습니다.
- **AI 도우미 제안**도 같은 방식으로 제목 기준 병합되어 기존 배치·색을 보존합니다.

## 내보내기가 4종으로 늘었습니다

| 형식 | 내용 |
| --- | --- |
| **PNG** | 현재 캔버스 이미지(2배 해상도) |
| **Excel** | 노드 표 — 담당자·부서·회당 파라미터, 통화별 비용 열과 숫자 서식 |
| **CSV** | 가져오기와 같은 14열 — 고쳐서 다시 가져오는 왕복 가능 |
| **Word** | 도형 순서도가 그려진 `.docx` — 노드 링크는 하이퍼링크로 |

## 운영 대시보드가 넓어졌습니다

- **설정 → 분석**의 대시보드에 운영 지표(맵·버전 상태·부서별 도입률·로그인/활동·누적 증가·최근 버전 이벤트·AI 사용량)와 기간 필터가 추가되었습니다.
- 대시보드 **열람 권한을 위임**할 수 있습니다 — 관리자가 사람·부서·유저그룹에 접근 권한을 부여하면 관리자가 아니어도 대시보드를 볼 수 있습니다.

## 그 밖에

- **자동 로그인**: 한 번 로그인하면 다음 방문부터 자동으로 로그인되고, 처음 열려던 화면으로 이어집니다.
- **노드 링크(URL)**: 노드에 외부 문서·시스템 링크를 달면 배지가 뜨고, 클릭해 미리보기·새 탭 열기를 할 수 있습니다. 표시 이름(라벨)도 지정할 수 있습니다.

문의나 불편 사항은 상단 **피드백** 버튼으로 보내주세요.
