## 2. 이번 리뷰 라운드 처리 내역

### F1 — 디시전 노드 드롭 트리거 명확화 (3분기)
노드를 **디시전 노드**에 드롭(출력 ≥1)하면 `EdgeDecisionModal`로 3택:
1. **분기 만들기** → `applyFlowEdges(rewire=false)`로 B→A 새 출력선 추가 → 자동 yes/no/기타 라벨 모달.
2. **인터셉트** → 출력선 **2개 이상이면 선택 모달**(`EdgeSelectModal`), **1개면 그 선에 바로** 끼워넣기(`interceptIntoEdge`, 라벨 보존).
3. **취소**.
- 비-디시전 노드는 기존대로(2개↑ 선택 / 1개 삽입·교체). `applyEdgeSelect`는 `interceptIntoEdge` 재사용으로 정리.

### F14 — `[`/`]` 끝/처음 클램프
하이라이트 노드 수가 변하지 않으면(경로 끝/처음 도달) reach를 증가/감소하지 않음 → `]`를 끝에서 더 눌러도 안 늘고, `[`가 바로 줄어듦(누른 만큼 되돌릴 필요 없음). Tab/Shift+Tab 포커스 이동은 그대로.

### ST — 맵 세팅 재구성
- **단일 스크롤 페이지**: 전 섹션(`<section id>`)을 위→아래로 나열, 탭 전환 없음.
- **좌측 앵커 내비**: 클릭 시 `scrollIntoView`, `IntersectionObserver`로 현재 섹션 하이라이트.
- **승인자 카드**: `aspect-square` 고정 크기(소속 개수 무관 동일) · 이름·아이디 + **소속 경로 최대 5단(센터/부서/팀/그룹/파트)** 표시, 아래 공간 확보.
  - 백엔드: `GET /maps/{id}/eligible-approvers`가 `EligibleApproverOut`(= 디렉터리 + `org_path`) 반환. `/directory` 응답 계약은 불변(서브클래스로 분리).

### DL — 삭제 안내 시각화
`DeleteMapDialog` — 휴지통 아이콘(에러 틴트) + 요점 3줄(휴지통 이동 / 7일 복구 / 영구삭제). 홈(MapDetailCard)·설정(DangerZone) 공용. 기존 글 위주 모달 대체.

### HM — 조직 아이콘 + 본인 표시
- 부서 행: 레벨별 아이콘(센터=Landmark, 담당=Building2, 팀=Users, 그룹=UsersRound, 파트=Boxes).
- 본인(현재 접속자 user 그랜트): 유저 아이콘 대신 **'me' 배지**(아이콘 동급 크기 원형).
