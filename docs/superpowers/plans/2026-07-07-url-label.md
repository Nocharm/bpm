# URL Label + Pill Input + Subprocess Designated URL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 노드 URL에 표시용 라벨을 추가해 액션 바 버튼 텍스트를 대체하고(호버 시 열기 아이콘), 인스펙터·노드 편집 모달에 공용 2행 필 편집기를 넣고, subprocess는 지정(designation) 어트리뷰트로 URL·라벨을 공급하며, CSV에 url_label 컬럼(URL 없는 라벨은 무시+서머리 표기)을 추가한다.

**Architecture:** `url_label`은 기존 `url`의 경로(모델→NodeIn→auto-augment→toAppNodes/buildGraph→CSV)를 그대로 복제. subprocess는 기존 sp_* 파이프라인(SubprocessDesignationIn→process_maps 컬럼→SubprocessRefOut→injectSubEnds 주입)에 `sp_url`/`sp_url_label`만 추가. 편집 UI는 신규 공용 컴포넌트 `UrlLabelField` 하나를 인스펙터·모달이 공유.

**Tech Stack:** FastAPI+SQLAlchemy+Pydantic(백엔드), Next.js 16+React 19(React Compiler)+@xyflow/react+Tailwind v4(프론트), pytest / vitest / playwright-core 스모크.

**Spec:** `docs/superpowers/specs/2026-07-07-url-label-design.md` (승인 + CSV 보정 반영)

## Global Constraints

- 삭제 규칙: **URL 삭제 시 라벨 동반 삭제**(UI + 서버 validator 이중 보장), **라벨만 단독 삭제 가능**.
- 라벨 있으면 액션 바 링크 버튼 = **라벨 텍스트만**(말줄임) + **hover 시 열기 아이콘 페이드-인**(레이아웃 밀림 금지). 라벨 없으면 기존(체인 아이콘 칩 + "Open link") 그대로.
- subprocess의 URL/라벨은 **지정 어트리뷰트**(`sp_url`/`sp_url_label`) — 호스트 맵에서 수정 불가(읽기전용 표시만), 노드 자체 url/url_label과 별개.
- 필 형식 = 담당자 칩과 동일: `rounded-sm border border-hairline bg-surface-alt px-1.5 py-0.5 text-fine` + `<X size={11} strokeWidth={1.5}/>`, 값은 `truncate`.
- 길이 제한: url 500 / url_label 100 (백엔드 Field·CSV MAX_LEN·input maxLength 모두 동일 값).
- 서버는 **길이만 검증**(스킴 검증은 클라이언트 — 기존 url 컨벤션). 지정 모달만 클라이언트에서 http(s) 검증(비어있지 않은데 `isHttpUrl` 불통과면 저장 비활성).
- CSV: URL 없는 행의 라벨은 **에러 없이 무시** + `ignoredLabelCount` 집계 → 임포트 전 서머리에 표기.
- i18n en/ko 동시 추가(tsc 강제). raw hex 금지 — 토큰만. React Compiler: effect 내 동기 setState 금지, deps 불일치 금지, **렌더 중 ref 변조 금지**(`react-hooks/refs` — 렌더 중 상태조정은 modal의 `prevNodeId` useState 패턴).
- 기존 자동저장(`updateSelectedData(patch, true)` / `scheduleAutoSave`) 경로 재사용, 신규 영속 로직 금지.
- 커밋마다 `PROGRESS.md` 한 줄 + 이 플랜 체크박스를 같은 커밋에 갱신. 커밋 메시지 `type(scope): English — 한국어`.

// URL 행 X = URL·라벨 동시 삭제, 라벨 행은 URL 있을 때만 노출되고 X = 라벨만 삭제.

import { X } from "lucide-react";
import { useState } from "react";

import { useI18n } from "@/lib/i18n";

const PILL_CLASS =
  "flex min-w-0 max-w-full items-center gap-1 rounded-sm border border-hairline bg-surface-alt px-1.5 py-0.5 text-fine text-ink";
const INPUT_CLASS =
  "min-w-0 flex-1 truncate rounded-sm bg-transparent px-1 py-0.5 text-right text-caption text-ink " +
  "hover:bg-surface-alt focus:bg-surface-alt focus:outline-none disabled:hover:bg-transparent";

export function UrlLabelField({
  url,
  urlLabel,
  readOnly,
  onChange,
}: {
  url: string;
  urlLabel: string;
  readOnly: boolean;
  onChange: (patch: { url?: string; urlLabel?: string }) => void;
}) {
  const { t } = useI18n();
  const [urlDraft, setUrlDraft] = useState("");
  const [labelDraft, setLabelDraft] = useState("");

    if (value) onChange({ url: value });
    setUrlDraft("");
  };
  const commitLabel = () => {
    const value = labelDraft.trim();
    if (value) onChange({ urlLabel: value });
    setLabelDraft("");
  };

  return (

              value={labelDraft}

              onKeyDown={(event) => {

      )}
    </>
  );
}
